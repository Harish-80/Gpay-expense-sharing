# Gpay-expense-sharing

First, install **Anaconda** and open **Jupyter Notebook**.
This project is a **GPay Expense Sharing program for friends** that calculates **total paid, total owed, and final balance**.

Install the required libraries using:
pip install numpy prettytable

Then import **NumPy** and **PrettyTable** in the code.

The friends in this project are **Rajesh, James, Aravind, and Harish**.
An **expense matrix** is created for 4 people using rows and columns.

When a friend pays an amount, the expense is shared among selected friends.
The program calculates each person's **balance and payment settlement**.

**PrettyTable** is used to display the results in **rows and columns** and suggest the final payment settlement.
